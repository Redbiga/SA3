# 介绍：
软件体系结果实验三：用二层CS架构和三层CS架构,BS架构完成一个简易的个人通讯录
# 使用到的技术：
主要是javaweb的技术，用mybatis实现对数据库的操作，使用传统的Swing和awt实现cs体系架构的客户端界面展示;
jsp和jquery，java servelet 实现BS体系结构中B的页面；
